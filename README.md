# BWR Services - Aircond Specialist 

A high-performance, responsive landing page for an air conditioning service business based in Malaysia. This project focuses on **conversion optimization**, **social proof**, and **seamless user communication** via WhatsApp integration.

## 🚀 Live Demo

https://bwr-services.vercel.app/

-----

## ✨ Key Features

### 🛠️ Interactive Booking Flow

  * **Step-by-Step UI:** A clean "How It Works" section guiding customers from inquiry to service completion.
  * **WhatsApp Integration:** Direct "Book Now" buttons with pre-filled custom messages to reduce friction for the customer.

### 🌟 Advanced Review System

  * **Infinite Vertical Scroll:** An automated, non-interruptible (on mobile) vertical scrolling feed for customer testimonials.
  * **Dynamic Proof of Work:** A legitimacy section that auto-cycles through "WhatsApp Verified" proof-of-work images with smooth cross-fade transitions.
  * **Interactive Lightbox:** Users can tap any review image or proof-of-work photo to view it in a full-screen, dimmed-background modal.

### 📱 Modern UI/UX

  * **Responsive Design:** Custom layouts optimized for Desktop (\>1000px) and Mobile.
  * **Adaptive Components:** The map and secondary legitimacy proofs automatically hide on smaller screens to keep the mobile experience fast and clutter-free.
  * **Glassmorphism:** Use of backdrop blurs and semi-transparent containers for a premium, modern aesthetic.

-----

## 🛠️ Tech Stack

  * **Framework:** [React.js](https://reactjs.org/) (Vite)
  * **Styling:** [Tailwind CSS](https://tailwindcss.com/)
  * **Icons:** [React Icons](https://react-icons.github.io/react-icons/) (FontAwesome)
  * **Animations:** Custom CSS Keyframes & Tailwind Transitions
  * **Deployment:** Vercel

## 🛠️ Installation & Setup

1.  **Clone the repository**

    ```bash
    git clone https://github.com/your-username/bwr-services.git
    ```

2.  **Install dependencies**

    ```bash
    npm install
    ```

3.  **Start the development server**

    ```bash
    npm run dev
    ```
